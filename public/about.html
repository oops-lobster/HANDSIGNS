손말관으로 돌아가기

SONMALGWAN · PROJECT NOTE

왜 이걸 만들었나요?

손말관은 완성된 통역사가 아니라, 극장에서 자막을 보듯 수어 영상의 흐름을 함께 확인하기 위한 프로토타입입니다.

언어 수어는 한국어의 보조가 아니라 하나의 언어입니다.

단어를 그대로 바꾸는 것만으로는 충분하지 않습니다. 수어는 어순, 표정, 공간, 맥락이 함께 움직이는 시각 언어입니다.

영상 기초 대화를 영상으로 바로 확인하는 경험이 필요했습니다.

“안녕하세요”, “이름이 어떻게 되세요?” 같은 표현을 입력하면 수어 단어 구조를 분석하고, 사전에 연결된 영상을 순서대로 보여주는 흐름을 목표로 합니다.

검토 전문가와 당사자의 피드백이 중심입니다.

이 MVP는 정답을 선언하는 도구가 아닙니다. 수어 전문가와 농인 당사자의 피드백을 받아 자연스러운 표현과 누락된 표현을 계속 보완하기 위한 실험입니다.

한계 수어사전 영상 조합은 완전한 통역을 대신하지 않습니다.

한국어와 수어는 다른 언어입니다. 손말관은 기초 표현을 확인하고 학습하는 보조 도구로 출발하며, 자연스러운 통역 품질은 전문가 검토가 필요합니다.

How it works

한국어 문장을 바로 영상으로 틀기보다, 먼저 수어 표현 단위로 정리합니다.

1. 한국어 입력

사용자가 일상 문장을 입력합니다. 예: “안녕하세요”, “홍길동입니다.”

2. Gemini 의미 분석

조사와 어미를 줄이고, 수어사전에서 찾을 수 있는 표준 표제어와 지문자 자모로 문장을 재구성합니다.

3. 수어 API 검색

분석된 토큰을 국립국어원 한국수어사전 기반 공공 API에서 찾아 영상 후보로 연결합니다.

4. 순서대로 자동 재생

찾은 영상을 수어 문장 구조 순서대로 이어서 재생해, 사용자가 직접 링크를 누르지 않아도 흐름을 볼 수 있게 합니다.

Expected Impact

손말관이 만들고 싶은 변화

Learn 수어를 배우는 계기

사용자는 한국어 문장이 어떤 수어 단어와 순서로 바뀌는지 영상으로 확인하며, 기초 표현을 반복해서 익힐 수 있습니다.

Talk 한국어가 익숙하지 않은 농인과의 의사소통

필담만으로 의미가 충분히 전달되지 않는 상황에서, 수어 영상 흐름을 함께 보여주며 대화의 실마리를 만들 수 있습니다.

Bridge 농인과 청인이 서로를 이해하는 접점

수어가 단순한 손동작이 아니라 표정과 맥락을 가진 언어라는 점을 경험하게 하고, 전문가 피드백을 통해 더 나은 방향으로 개선합니다.

MVP Scope

지금은 “기초 대화 검증”에 집중합니다.

현재 잘 맞추고 싶은 것
  • 인사, 자기소개, 이름 묻기 같은 기초 일상 대화
  • 유치원/초등 저학년 수준의 기본 어휘 확인과 반복 학습
  • 수어사전에 있는 표준 단어 영상 자동 연결
  • 사람 이름처럼 사전에 없는 고유명사의 지문자 자모 분해
  • 전문가가 피드백하기 쉬운 수어 문장 구조 표시
아직 조심해야 하는 것
  • 자연스러운 수어 통역 전체를 대체하지 않습니다.
  • 한국어의 모든 단어를 수어사전 영상으로 일대일 치환할 수 없습니다.
  • 표정, 속도, 공간 활용은 영상 조합만으로 완전히 표현하기 어렵습니다.
  • 사전에 없는 표현은 누락되거나 자모로 분해될 수 있습니다.
  • Gemini와 공공 API의 응답 상태에 따라 결과가 달라질 수 있습니다.
