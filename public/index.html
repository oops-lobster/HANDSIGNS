손말관

마이크나 문장으로 입력하면 분석된 수어 영상을 순서대로 재생합니다.

입력 대기 준비됨 영상 재생
글자 크기
손말관은 한국어 문장을 수어사전 영상 순서로 재생하는 프로토타입입니다.
  1. 01말하거나 문장 입력
  2. 02수어 표현으로 분석
  3. 03영상이 순서대로 재생
말하거나 입력하면 순서대로 재생됩니다
마이크를 누르면 음성이 문장으로 입력됩니다.

상영관 안내: 입력 문장은 국립국어원 한국수어사전 표제어 중심으로 정리해 영상 순서로 재생합니다.

손말관 상영 대기 중 문장을 수어 표현 단위로 정리하고, 국립수어사전의 영상 정보를 찾아 자막처럼 순서대로 재생합니다.
